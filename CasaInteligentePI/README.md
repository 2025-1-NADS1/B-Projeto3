# Testes de Comunicação em Rede

## Objetivo
Verificar se a comunicação entre cliente e servidor funciona corretamente.

## Testes Implementados

- Conectividade entre cliente e servidor
- Envio de mensagem do cliente para o servidor
- Resposta do servidor para o cliente

## Como Executar

### 1. Compile os arquivos

```bash
csc Server.cs
csc Client.cs
